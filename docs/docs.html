ZIPPY

Documentação da ZIPPY

Visão Geral

A ZIPPY é uma interface simples e eficiente para interação com uma inteligência artificial, projetada para oferecer respostas rápidas e precisas.

Utilizando a API do Gemini, a ZIPPY integra funcionalidades avançadas, incluindo pesquisa na web em tempo real, garantindo que as respostas estejam sempre atualizadas e relevantes.

A interface é intuitiva, responsiva e acessível, facilitando a navegação e a comunicação fluida entre o usuário e a IA.

Funcionalidades Principais

  • Pesquisa na Web em Tempo Real: Permite que a IA acesse informações atualizadas diretamente da internet para fornecer respostas mais precisas e relevantes.
  • Resposta Rápida: A ZIPPY processa e responde às mensagens com agilidade, proporcionando uma experiência fluida e eficiente.
  • Facilidade de Movimentos: A interface é projetada para ser simples e intuitiva, com navegação fácil e controles acessíveis, garantindo conforto no uso diário.
  • Compreensão Avançada: A IA entende o contexto das mensagens, permitindo conversas naturais e coerentes.
  • Alternância de Tema: O usuário pode escolher entre tema claro e escuro, com a preferência salva para futuras sessões.
  • Gerenciamento de Histórico: Histórico de conversas armazenado localmente, com opção de limpeza pelo usuário.
  • Notificações Intuitivas: Mensagens de alerta e confirmação são exibidas de forma clara e discreta.

Estrutura da Página

  • Cabeçalho: Exibe o nome da IA "ZIPPY" com ícone e botão para abrir configurações.
  • Área de Mensagens: Espaço principal para exibição das mensagens trocadas entre usuário e IA.
  • Formulário de Entrada: Campo para digitar mensagens, botão para ativar pesquisa na web e botão de envio.
  • Modal de Configurações: Permite limpar histórico, alternar tema e desconectar da conta.
  • Notificações: Área para exibir mensagens temporárias ao usuário.

Como Usar

  1. Faça login na aplicação.
  2. Digite sua mensagem no campo inferior e envie pressionando Enter ou clicando no botão de envio.
  3. Ative a pesquisa na web clicando no ícone do globo para obter respostas com dados atualizados.
  4. Visualize as respostas da IA aparecerem com animação de digitação.
  5. Acesse as configurações para limpar histórico, alternar tema ou desconectar.

Acessibilidade e Usabilidade

  • Uso de roles ARIA para descrever regiões e estados.
  • Labels e descrições para botões e campos de entrada.
  • Gerenciamento de foco ao abrir e fechar modais.
  • Suporte a navegação por teclado e atalhos (ex: Enter para enviar).
  • Contraste adequado para temas claro e escuro.
